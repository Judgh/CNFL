# This is the repository for the paper "Integrating LLM-Based Code Naturalness For Effective Software Fault Localization".

---

This paper provides a unified framework for integrating code naturalness evaluation with existing fault localization approaches.
It further empirically investigates the role of code naturalness in fault localization, studying when, why and how code naturalness can properly support fault localization.
